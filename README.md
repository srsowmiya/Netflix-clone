Netflix Clone with Mood-Based Recommendations

A Netflix-inspired web application built using React ,Firebase and Fast API featuring authentication, video playback, and an AI-inspired mood-based recommendation system using curated content.

Features
User authentication (Login & Signup) using Firebase
Video playback using YouTube trailers
Mood-based movie recommendations
Fully responsive Netflix-style UI
Fast performance with Vite
Deployed on Vercel

Mood-Based Recommendation System
Users can enter their current mood (happy, sad, excited, stressed, etc.), and the application recommends movies accordingly using pre-curated movie lists.
This approach simulates AI behavior while keeping the application fast, reliable, and deployment-friendly.
